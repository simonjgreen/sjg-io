This site is built with modern tools and practices. Here's the technical stack:

## Technology

- **Astro** - Static site generator with component islands
- **TypeScript** - Type safety and better developer experience  
- **MDX** - Content authoring with React components
- **Tailwind CSS** - Utility-first CSS framework
- **Content Collections** - Type-safe content management

## Deployment

- **Cloudflare Workers** - Static site hosting
- **GitHub Actions** - CI/CD pipeline
- **Wrangler** - Cloudflare Workers CLI

## Performance

Built for speed:
- Static site generation
- Minimal JavaScript
- CDN delivery via Cloudflare
- Optimised images and assets

## Accessibility

- Semantic HTML structure
- Proper heading hierarchy
- Keyboard navigation support
- Screen reader compatibility

## Analytics

- **Plausible Analytics** - Privacy-focused, no cookies
- No personal data collection
- GDPR compliant

Uses system fonts for optimal performance and compatibility.
